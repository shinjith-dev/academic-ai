
# Academic AI

A react-native app for Academic AI project

## Commands

Run in development mode

```bash
  # android
  npm run android

  # ios
  npm run ios

  # web
  npm run web
```

Run in production mode

```bash
  # run below command to clean existing builds
  # only needed in major changes and config updates
  # npm run clean 

  # generate production ready app
  # android
  npm run build:android
```
